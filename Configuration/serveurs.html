Service SSH

SSH (Secure Shell) permet d'établir une connexion sécurisée à distance sur la machine virtuelle. Pour installer le service SSH, la commande suivante a été exécutée : 


sudo apt install openssh-server

Une fois installé, le service SSH permet de se connecter à la VM en utilisant son adresse IP et un client SSH (comme PuTTY ou la commande ssh).

Création de répertoires

Pour organiser les fichiers du projet et préparer l’hébergement du site, des répertoires spécifiques ont été créés sur la machine virtuelle avec les commandes suivantes : 


mkdir /var/www/html/presentation

Ce répertoire contient les fichiers HTML, CSS, et autres ressources nécessaires pour le site web.
