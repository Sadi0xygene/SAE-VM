Développement et Outils

Configuration de WebStorm

WebStorm est l’éditeur utilisé pour développer le site web de ce projet. Voici les étapes pour configurer WebStorm :

  • Télécharger et installer WebStorm depuis le site officiel de JetBrains.
  • Ouvrir le répertoire contenant les fichiers du projet (/var/www/html/presentation).
  • Configurer le chemin du projet dans WebStorm pour faciliter l'accès aux fichiers.

Développement du site

Le site a été développé en utilisant les technologies suivantes :

  • HTML5 : pour structurer les pages du site.
  • CSS3 : pour le design et le style des pages.
  • Bootstrap : pour créer des éléments responsives rapidement.

L'architecture du site inclut une page d'accueil et plusieurs pages organisées par section (Concepts, Configuration, Serveurs, etc.). Le code HTML et CSS a été testé à chaque étape pour garantir une expérience utilisateur fluide.
