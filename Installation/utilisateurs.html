Gestion des groupes

Pour regrouper les utilisateurs, nous avons créé un groupe "BinomeG7" à l'aide des commandes suivantes : 


sudo groupadd binomeG7
sudo usermod -aG binomeG7 binomeG7_1
sudo usermod -aG binomeG7 binomeG7_2

Commande Prompt

Le prompt est la ligne qui s'affiche dans le terminal pour indiquer que le système est prêt à recevoir des commandes. Nous avons personnalisé le prompt en modifiant le fichier .bashrc avec la commande suivante : 


export PS1="[\u@\h \W]$ "
